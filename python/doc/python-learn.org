* object and types
python uses object to store data, each object has its type.

** 5 types
** string (str)
   use 'a' or "a"
** integer (int)
   21(dec), 025(oct), 0x15(hex)
** float 
   1.48, 21.0, 21., .21, 2.1e2
** boolean (bool)
   True, False
** complex
   1+1j

** check type
  type(True), type("aa"), type(21.5)
